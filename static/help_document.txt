# ESCUDE-CG-Composer 操作说明

## 概述
ESCUDE-CG-Composer 是一个用于加载、解读和合成CG或立绘的工具。用户可以通过选择包含lsf文件与组件图片的CG或立绘解包目录，查看和编辑CG或立绘，并将其保存到指定目录。

## 基本操作

### 1. 导入目录
- 点击菜单栏中的“导入目录” -> “导入目录”。
- 选择包含lsf文件与组件图片的CG或立绘解包目录。
- 程序将读取目录中的所有lsf文件，并在侧边栏显示。

### 2. 查看CG或立绘
- 在侧边栏中点击某个lsf文件名。
- 程序将显示该lsf文件对应的CG或立绘，并在底边栏展示组成CG或立绘的所有组件图片。

### 3. 调整图片差分
- 在底边栏中点击某个组件图片的左右箭头按钮。
- 程序将根据按钮点击调整对应的图片差分，并更新显示的CG或立绘。

### 4. 保存CG或立绘
- 点击菜单栏中的“提取” -> “提取图片”。
- 程序将弹出保存对话框，用户可以选择保存路径和文件名。
- 程序将合成的图片保存到指定目录，并显示成功提示。

## 注意事项
- 确保导入的目录包含正确的lsf文件和组件图片。
- 在调整图片差分时，程序会实时更新显示的CG或立绘。
- 保存图片时，程序会自动处理文件名重复的情况。
